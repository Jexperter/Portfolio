SMPC Wallet

The aim of this project is to develop a robust and secure crypto wallet application leveraging Secure Multi-Party Computation (SMPC), requiring both the user and the service provider for accessing the wallet. It offers features enabling users to securely sign up, log in, manage their wallets, view balances, and send transactions. The system architecture comprises three main components: frontend, backend, and database, each serving a specific role.
