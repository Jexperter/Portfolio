Dissertation

This dissertation details the development and implementation of a secure cryptocurrency wallet application leveraging Secure Multi-Party Computation (SMPC). SMPC is a cryptographic method that allows multiple parties to jointly compute a result without revealing their individual inputs to each other. The primary goal of the project was to enhance the security of private keys while maintaining a seamless user experience.
