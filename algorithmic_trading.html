Description

I have developed several scripts to collect data, including past trades and liquidation events, to effectively backtest my chosen strategies. I'm currently working on implementing a strategy that takes trades when a certain amount of liquidation occurs, with added parameters and conditions to maximize returns, improve the Sharpe ratio, and reduce maximum drawdown.

This project represents my ongoing effort to refine my strategies by leveraging data and analytics. Feel free to reach out for advice or collaboration at: akterjesan@gmail.com.

Current Strategy

This is a simple liquidation strategy where it takes a trade when a certain amount of liquidations occur. I used historical trade and liquidation data to simulate the outcomes and measure the effectiveness of the strategy.

Check it out: Liquidation Backtest.

Fetch Asset History

This script provides historical data trades from any ticker, interval, start date, and end date using the Binance API. I usually use this to generate datasets that I later use to backtest strategies.

Check it out: Historical Crypto Data.
